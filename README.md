# CatsMap 🐱

**Keyboard Adaptation Trainer** — a browser-based typing tool to help retrain muscle memory when switching keyboard layouts or adapting to a new keyboard.

**Live demo:** [colubambi.github.io/catsmap](https://colubambi.github.io/catsmap)

## What it does

CatsMap helps you identify and drill your problem keys when switching keyboards or layouts. Instead of generic typing tests, it focuses on the keys you actually struggle with — the ones your fingers keep reaching for in the wrong place.

- **Problem key selection** — mark the keys giving you trouble
- **Cat Attack Mode** — timed drills targeting your weak spots
- **Error heatmap** — visual breakdown of your most-missed keys
- **Progress tracking** — see improvement across sessions
- **Keyboard status view** — real-time feedback on key accuracy

## Tech

Single-file HTML/CSS/JS — no dependencies, no build step. Open `index.html` in any browser.
